British Denim

Sitio web ecommerce enfocado en moda denim con estética premium y diseño moderno.  
Proyecto desarrollado con HTML, CSS y JavaScript, priorizando experiencia visual, animaciones suaves y estructura escalable a Django.

Características

- Hero full screen con animación sutil
- Header fijo con transición al hacer scroll
- Grid de productos responsive
- Página de perfil de usuario (post-login)
- Diseño completamente responsive
- Animaciones on-scroll
- Menú móvil animado
- Footer estructurado tipo ecommerce premium

Tecnologías Utilizadas

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla)
- Responsive Design (Media Queries)
- Animaciones CSS

Funcionalidades Implementadas

Header Inteligente
- Transparente sobre el hero
- Cambia a fondo blanco al hacer scroll
- Transición suave
- Sombra sutil

Hero Section
- Imagen fullscreen
- Overlay oscuro para contraste
- Animación de zoom lenta
- Tipografía fluida con `clamp()`

Sección de Productos
- Grid adaptable con `auto-fit`
- Cards con efecto hover
- Diseño limpio y moderno

Perfil de Usuario
- Nombre
- País
- Contador de productos registrados
- Listado de referencias con:
  - Descripción
  - Fecha de registro
- Botones:
  - Editar perfil
  - Cerrar sesión

Responsive

Breakpoints principales:

- ≤ 991px → menú colapsable
- ≤ 575px → footer en columna

Optimizado para:

- Mobile
